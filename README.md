# 🚀 AI Career Coach

An AI-powered Career Assistant built using **Google ADK**, **MCP**, **FastAPI**, and **React** to help users throughout their job search journey.

From resume analysis to interview preparation, AI Career Coach provides personalized guidance using a team of specialized AI agents.

---

# 🌟 Features

- 📄 AI Resume Analysis
- 🎯 ATS Resume Score
- 💼 AI Job Search
- 📝 AI Cover Letter Generator
- 🎤 Interview Preparation
- 🧠 Career Roadmap Generator
- 📊 Salary Insights
- 🏢 Company Research
- 📅 Application Tracker
- 💬 AI Career Chat Assistant
- 🔒 Secure Authentication
- 🛡️ Scam & Fraud Detection

---

# 🤖 Multi-Agent Architecture

The platform consists of multiple AI agents working together.

- 🧠 Orchestrator Agent

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/ai-career-coach.git

cd ai-career-coach
```

---

## Frontend

```bash
cd frontend

npm install

npm run dev
```

---

## Backend

```bash
cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python -m uvicorn app.main:app --reload
```
